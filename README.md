# Sign-up Form

A clean, modern **Sign-up Form** built with **HTML, CSS, and JavaScript**, featuring a two-panel layout with a full-height image banner and a polished, professional form design.

This project was built as part of [The Odin Project Intermediate HTML & CSS Course](https://www.theodinproject.com/paths/full-stack-javascript/courses/intermediate-html-and-css).

---

## Overview

This is a **responsive sign-up form interface** modeled after real web authentication pages.

- Split-screen layout with a **left image panel** and **right form panel**.  
- Fully structured form with inputs for **name, email, phone, and password**.  
- Live **focus styling** and input feedback.  
- Modern banner overlay with a centered company name.  
- “Create Account” button with hover and click animations.  

The project focuses on **clean form structure, layout fundamentals, and polished UI styling**.

---

## Features

- Multi-column form rows that adjust cleanly across screen sizes.  
- Label + input grouping for clarity and accessibility.  
- Focused input highlighting with custom border and shadow effects.  
- Fully responsive layout using flexbox.  
- Image banner with semi-transparent overlay for text readability.  
- Professional, minimal color palette with subtle animations.

---

## Technologies Used

- **HTML5** — semantic page structure  
- **CSS3** — layout, animations, and responsive design  
- **JavaScript (ES6)** — prepared for form validation and interactivity  
- **System UI fonts** — clean, modern base typography  

---

## Styling

- Split layout with a **full-height left image panel**.  
- Banner text with a dark translucent overlay across the image.  
- Soft shadowed form container with rounded edges.  
- Smooth input focus transitions for improved UX.  
- Professional button animations with lift and press feedback.

## Development Highlights

- Structuring a clean multi-column form using flexbox.
